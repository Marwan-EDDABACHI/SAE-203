# SAÉ 203 - Site Web BUT MMI

Projet réalisé dans le cadre de la SAÉ 203 : "Concevoir un site web avec une source de données". Ce site est une plateforme dynamique pour le BUT MMI de l'IUT de Cergy-Pontoise.

## Description du Projet
Ce projet consiste en la réalisation d'un site web dynamique utilisant PHP et MySQL pour la gestion des données (articles, auteurs, messages). Il comprend une partie front-office et une partie back-office (administration) pour la gestion du contenu.

## Sécurité et Accès à l'Administration
**Note importante concernant la sécurité :**
Conformément aux bonnes pratiques de développement, les identifiants de connexion réels à la base de données et à l'espace d'administration ne sont pas inclus dans ce dépôt public pour des raisons de sécurité. 

Vous trouverez les identifiants nécessaires pour accéder à l'administration et tester l'application directement dans le **fichier de que j'ai déposé sur Moodle**.

## Technologies Utilisées
* **Langages :** HTML5, CSS (avec utilisation de `rem`), PHP, SQL (MySQL).
* **Framework / Styling :** CSS personnalisé et Tailwind CSS (dans le back-office).


## Installation
1. **Base de données :** Importez le fichier `base-de-donnees.sql` présent à la racine dans votre interface PhpMyAdmin.
2. **Configuration :** - Dupliquez les fichiers `.env.dev` ou `.env.prod` selon votre environnement.
   - Configurez les variables `NOM_BDD`, `SERVEUR_BDD`, `UTILISATEUR_BDD` et `MDP_BDD` avec vos accès locaux ou distants.
